# 邱御恩的 115 學年度二年級重點科目教材統計表

本清單依據 **桃園市潮音國民小學 115 學年度二年級教科書版本** 整理，僅列出需要額外購買教材的重點科目，方便購書時快速核對統計。

---

### 📊 購買品項總計
*   **自修（共 2 本）**：國語（康軒）、數學（翰林）
*   **評量（共 3 本）**：國語（康軒）、數學（翰林）、生活（南一）
*   **測驗卷（共 3 份）**：國語（康軒）、數學（翰林）、生活（南一）
*   **額外練習（共 2 本）**：數學計算練習本（翰林）、國語生字語詞簿/寫字練習本（康軒）

---

### 📋 重點教材統計明細表

| 科目 | 學校版本 | 建議購買教材 | 功能用途說明 |
| :--- | :--- | :--- | :--- |
| **國語文** | **康軒版** | 自修、評量、測驗卷、生字語詞簿 | 字詞觀念、生字筆順美字練習、單元複習、段考前模擬衝刺。 |
| **數學** | **翰林版** | 自修、評量、計算練習本、測驗卷 | 解題觀念、加減法與九九乘法熟練、段考前限時答題模擬。 |
| **生活課程** | **南一版** | 評量、測驗卷 | 整合社會、自然、藝術領域的課後檢測，段考前複習與考卷模擬。 |

*註：其餘未列出科目（如英語、本土語、健體等）均免額外自購教材。*
